Class: Chicken

Chicken(xPos, onReadyCallbackopt)

Represents a walking chicken enemy. Inherits animation and positioning logic from MovableObject.

Constructor

new Chicken(xPos, onReadyCallbackopt)

Creates a new Chicken instance at a given X position. Loads animation images and starts the walking animation with random speed.
Parameters:

xPos number The horizontal position where the chicken starts.
